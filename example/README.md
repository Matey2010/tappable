# Tappable Example App

This example app demonstrates all features of the `tappable` package.

## Running the Example

```bash
cd example
flutter run
```

## What's Included

- **Tappable Widget Demos**: Basic usage and customization examples
- **TappableContainer Demos**: Container-based tappable widgets with decorations
- **Tap Mechanics Showcase**:
  - Individual mechanics (Ripple, Scale, Press, Opacity, None)
  - Combined mechanics (Ripple + Scale, Press + Opacity, etc.)
  - Custom configurations
- **Real-world Examples**: Practical use cases like buttons, cards, and interactive elements

## Features Demonstrated

- Material ripple effects
- Scale animations
- 3D press effects
- Opacity/fade animations
- No-effect tappables
- Combining multiple mechanics
- Custom styling and decorations
- Disabled states
